# Sales Performance Dashboard – Excel

## Overview
This project presents an interactive Excel dashboard designed to analyze sales performance across regions, categories, and time periods. The dashboard helps in understanding key business metrics and supports data-driven decision-making.

## Tools Used
- Microsoft Excel / WPS Excel
- Pivot Tables and Pivot Charts
- Excel formulas and calculations
- Slicers for interactive filtering
- Dashboard design and formatting

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin (%)
- Total Orders
- Average Order Value (AOV)

## Analysis Covered
- Monthly sales trend (Jan 2023 – May 2024)
- Region-wise sales performance
- Sub-category-wise sales and profit analysis
- Year, region, and category-based filtering using slicers

## Dataset
- Simulated sales dataset
- Time period: January 2023 to May 2024
- Includes sales, profit, order, region, category, and sub-category data
